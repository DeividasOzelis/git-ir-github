Git

Git - populiariausia kodod versijvimo programa.

Git komandos

  • git init - sukuria gito repozitorija dabartiniame aplankale.
  • status - nurodo info apie lokalios gito repozitorijos statusa.
  • add - prideda nurodyta faila prie commit'inimui paruostu failu(staged).(add . - prideda viska esamoj direktorijoj ir nereikia nurodinet kiekvieno failo atskirai)
  • rm --cached - isima nurodytus failus is stage'inimo vietos.
  • commit -m "zinute" - sukuria kodo versija is steidzintu failu ir nurodo zinute kas buvo keista kode, kokia versija ar panasiai
  • restore - grazina nurodytus failus i buvusio komito busena.
  • push - is lokalios repozitorijos nusiuncia commit'us i isorine repozitorija
  • pull - is isorines repozitorijos atsiuncia i lokalia repozitorija
  • clone - clonuoja nurodyta repozitorija is isorines i lokaliagit
  • log - isveda commit'u sarasa
  • checkout + commitID/branchName - pereina i nurodyta kodo versija ARBA i kita branch'a
  • branch - naujo branch'o sukurimas
